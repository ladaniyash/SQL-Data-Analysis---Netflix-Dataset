# SQL-Data-Analysis---Netflix-Dataset

📌 Overview
This project focuses on analyzing the Netflix dataset using SQL (MySQL) to extract meaningful insights about content trends, genres, and distribution. The analysis involves writing 200+ SQL queries covering data cleaning, exploration, and business-driven questions.

🛠️ Tools & Technologies
MySQL
SQL (Joins, Aggregations, Subqueries, CASE statements)
Excel (optional for visualization)

📂 Dataset
Dataset: Netflix Titles Dataset
Contains information such as:
Title
Type (Movie/TV Show)
Director & Cast
Country
Release Year
Rating
Duration
Genre

🔍 Key Insights

🎬 Content Distribution
Movies dominate the platform (~70% of total content)
TV Shows make up a smaller portion
👉 Insight: Netflix focuses more on movie content than series

🌍 Country-wise Content
United States has the highest number of titles
India is among the top contributors
👉 Insight: Netflix is expanding strongly in global markets

📅 Release Trends
Content growth increased rapidly after 2015
Peak content production between 2018–2020
👉 Insight: Reflects Netflix’s expansion and original content strategy

🎭 Popular Genres
Top genres: Drama, Comedy, Documentary
👉 Insight: Drama is the most dominant genre globally

⏱️ Duration Analysis
Most movies are between 80–120 minutes
TV Shows usually have 1–2 seasons
👉 Insight: Preference for shorter, binge-friendly content

⭐ Ratings Analysis
Most content is rated TV-MA and TV-14
👉 Insight: Netflix mainly targets mature and teenage audiences

📈 Content Addition Trends
Huge spike in content addition during 2019–2020
👉 Insight: Rapid expansion followed by stabilization

🎥 Data Quality Observation
Missing values found in director and cast columns
👉 Insight: Real-world datasets require data cleaning before analysis

❓ Business Questions Solved
Which country produces the most Netflix content?
What are the most popular genres?
How has content production changed over time?
Movies vs TV Shows – which dominates?
What are the most common ratings on Netflix?


🧠 SQL Skills Demonstrated
Data Retrieval & Filtering
Used SELECT, WHERE, IN, BETWEEN, LIKE, and logical operators to extract specific insights from the dataset
Data Cleaning & Handling Missing Values
Identified and handled missing data using IS NULL and IS NOT NULL conditions
Sorting & Limiting Data
Applied ORDER BY and LIMIT to organize and retrieve top/bottom records efficiently
Aggregate Functions
Used functions like COUNT(), MAX(), MIN(), and AVG() to summarize data and generate insights
Grouping & Aggregation
Performed data segmentation using GROUP BY and filtered grouped results with HAVING
Distinct Analysis
Applied DISTINCT to identify unique values across multiple columns
Conditional Filtering
Used complex conditions with AND, OR, and NOT to solve business queries
Pattern Matching
Implemented wildcard searches using LIKE for text-based analysis
Multi-column Analysis
Analyzed relationships between columns using combined grouping (e.g., type, country, rating)

📊 Conclusion

This project demonstrates practical SQL skills by solving real-world business problems using the Netflix dataset. It highlights trends in content production, audience targeting, and platform strategy.
